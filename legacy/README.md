# Legacy

Ce dossier regroupe les anciens prototypes de **Neural Stock Exchange**.

Ils sont conserves pour leur valeur historique et pedagogique, mais ne representent pas le futur flux stable du projet.

Statuts principaux :
- `dow_macro/` : ancien prototype Dow Jones + macro, lie a des fichiers Rev2 perdus ;
- `gold/` : ancien prototype Gold, recuperable et prioritaire pour `nse-engine-gold` ;
- `backend/` : ancien backend Flask casse, conserve comme trace ;
- `dashboard/` : ancien dashboard React placeholder, en attente ;
- `notebooks/` : ancien notebook exploratoire.

Regle :
le code legacy peut etre lu, documente et reutilise comme reference, mais le futur code public doit vivre dans `src/nse_engine/`.
