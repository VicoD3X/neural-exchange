# Roadmap post-GitHub

Cette note remplace l'ancien fichier temporaire `POST_GITHUB_NEXT_AUDIT.txt`.

## Objectif

Faire evoluer Neural Stock Exchange vers un laboratoire experimental plus robuste, sans le transformer en produit financier.

## Axes de progression

1. Evaluation Rev4 avancee : walk-forward, baselines supplementaires, erreurs par regime.
2. Signal de stress : seuils alternatifs `Panic_Mode`, precision / recall, faux positifs et faux negatifs.
3. Flux multi-marches : Dow Jones, NASDAQ, puis comparaison multi-source.
4. Visualisation locale : Streamlit pour explorer les rapports Rev4.
5. Vitrine publique : dashboard React/Vite statique sur GitHub Pages.

## Regles de positionnement

- Aucun conseil financier.
- Aucun bot de trading.
- Aucune promesse de prediction fiable.
- Les dashboards lisent des exports, ils ne recalculent pas le pipeline.
- La valeur du projet reste : data engineering, PyTorch, evaluation critique, reporting et transparence.
