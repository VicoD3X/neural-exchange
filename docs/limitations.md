# Limites et non-usage

Neural Stock Exchange est un projet experimental.

## Limites techniques

- Pas de validation walk-forward avancee.
- Pas de comparaison robuste contre toutes les baselines pertinentes.
- Pas de garantie de stabilite hors periode etudiee.
- Rev2 et Rev2.5 ne sont pas completement reproductibles.
- Rev3 n'est pas un pipeline actif.
- Le backend Flask historique est archive.
- Le dashboard React historique n'est pas maintenu.

## Limites de donnees

- Les donnees de marche et macro sont reconstruites depuis des sources tierces.
- Les API et exports publics peuvent changer.
- Les donnees locales completes ne sont pas versionnees.
- Les anciens CSV legacy ne sont pas utilises comme reference active.

## Non-usage financier

Ce projet ne fournit pas :

- conseil financier ;
- signal de trading ;
- recommandation d'achat ou de vente ;
- systeme de prediction fiable ;
- outil de gestion de risque en production.

Toute lecture des resultats doit rester retrospective, pedagogique et experimentale.
