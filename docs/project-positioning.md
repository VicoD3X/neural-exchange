# Neural Stock Exchange - Positionnement

Neural Stock Exchange est un laboratoire experimental de signaux de marche.

Le depot garde volontairement une dimension de projet etudiant experimental : il montre l'evolution d'un prototype personnel fragmente vers une base plus propre, plutot qu'un produit financier finalise.

Le projet explore :
- des series temporelles financieres ;
- des features macroeconomiques ;
- des indicateurs de volatilite ;
- des modeles LSTM PyTorch ;
- des signaux de stress de marche.

Le projet ne doit pas etre presente comme :
- un bot de trading ;
- un conseil financier ;
- une prediction fiable du marche ;
- une application de production.

Le flux actif propre est Rev4.
Rev2 et Rev2.5 restent des traces legacy.
Rev3 reste une archive avancee, utile uniquement comme inspiration technique.

La valeur du projet vient autant de la remise en ordre que du modele lui-meme : separation du legacy, regeneration des donnees, tests offline, rapports reproductibles et comparaison honnete face a des baselines naives.
